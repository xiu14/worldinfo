## English

### Uninstall Old Version
1. Open the **Extensions** panel in SillyTavern
2. Find the **World Info Recommender** extension
3. Click the **Uninstall** button
4. **Restart SillyTavern**

### Install New Version (with language toggle)
1. Open the **Extensions** panel
2. In the **Install Extension** input field, enter:
   ```
   https://github.com/xiu14/SillyTavern-WorldInfo-Recommender
   ```
3. Click **Install** button
4. Wait for installation to complete
5. **Restart SillyTavern**
